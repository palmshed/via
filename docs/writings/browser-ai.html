Via and AI

How AI works inside Via.

Page overview

One-click page overview summarizes the current tab so you can decide what to read next.

Explanations

Highlight text for explanations. Select a passage and ask for context without leaving the page.

AI chat

Ask in AI chat about the current page, follow up, and generate quick summaries when AI is configured.

Providers

AI requests use the configured provider services. Connect your own Firebase project from the settings panel to enable AI features.
