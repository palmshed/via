# Releasing Via

This document details the release automation workflow for Via.

## Release Automation

[Release 1.28.9](https://github.com/palmshed/via/releases/tag/desktop/app-1.28.9) was the last desktop release published directly by GitHub Actions.

Current workflow automation uses the [Palmshed Via GitHub App](https://github.com/apps/palmshed-via) for repository automation, including releases, project workflows, and maintenance tasks.
